# Shortest-route-for_single_source
I made the best shortest route finder from source to destination. This is a simple shortest route to find 2D map project with PHP and javascript.
# Language
- Java Script
- PHP
- HTML
- CSS
# Algorithm
- Depth First Search (DFS)
